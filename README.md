## Key Features

* Removes: 
  - single character words
  - doug specific strings like "rigged", "doug", "morb" etc.
  - number-only messages
  - mentions
  - donation/resub messages
  - spamming of a single word
  
* Works with live streams
* Possible to enable for ALL VODs
